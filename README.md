The purpose of this project is to provide a simple captive portal
solution for Debian distribution (firstly designed for raspberrypi).

# Depedencies 

- apache2
- iptables
- php5
- conntrack

# Requirements

The portal user and www-data must be sudoers for :
- iptables
- conntrack
- arp
